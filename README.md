# ⚽ Soccer Manager

**Soccer Manager** is a modern web application built for groups of friends who want to easily organize and keep track of their football matches. The app allows you to set up upcoming matches, divide players into balanced teams, record final scores, and maintain a history of everyone's statistics.

## 🚀 Key Features

- **Complete Authentication System**: Registration and Login securely handled via Supabase Auth.
- **Manager Dashboard**: Overview of team statistics and match results.
- **Profile Management**: User area with customization options and secure account deletion.
- **Responsive Design**: Interface fully optimized for both desktop and mobile devices using Tailwind CSS.

---

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Email Provider**: [Brevo](https://www.brevo.com/) (SMTP Relay)
- **Hosting**: [Vercel](https://vercel.com/)
